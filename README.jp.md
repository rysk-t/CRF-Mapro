CRF-Mapro
=========

# 概要
[Processing](http://processing.org) によるClassical Receptive field のマッピング用プログラムです．

# 操作
 - マウスクリックしながら数字[1-4]: モード変更
 
    * 1: ランダムドット
    * 2: ランダムバー
    * 3: ランダムドット(円形モード)
    * 4: バー


 - b: 刺激拡大
 - s: 刺激縮小
 - n: ドット，バー数増加
 - w: 刺激幅変更
 - h: 刺激高さ変更
 - o: ランダムバー，バーモードでの方位変更
 - r: 刺激パラメータリセット
 - ESC: 終了
